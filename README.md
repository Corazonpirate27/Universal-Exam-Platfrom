Universal Exam Platform 2.0 transforms static study materials (PDFs/audio) into an interactive, computer-based testing environment.

The IELTS mode now simulates a British Council style computer-delivered test experience with:

- Listening, Reading, Writing, and live-examiner Speaking practice areas
- split-screen passage/question workflow
- question palette, next/previous navigation, answered status, and review flags
- on-screen timer, audio controls, volume control, notes, highlighting, and autosave
- Writing Task 1/Task 2 editor with automatic word count and copy/paste tools
- final review dialog before mock submission
- Acrobat-style PDF toolbar with fit width, fit page, zoom, page jump, rotate, PDF focus, and exam lock mode
- persistent PDF highlights that survive zoom, page re-rendering, autosave, and exported session files
- IELTS-style answer controls including text completion, multiple choice, TRUE/FALSE/NOT GIVEN, and matching selectors
